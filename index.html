Documentacao

Guia completo de instalacao e uso

1

Instalar Dependencias

Instale todas as dependencias Python necessarias:

pip install -r requirements.txt

Certifique-se de ter Python 3.8+ instalado

2

Configurar Credenciais

Configure suas credenciais do Google Calendar:

# Criar arquivo .env na raiz do projeto
GOOGLE_CLIENT_ID=seu_client_id_aqui
GOOGLE_CLIENT_SECRET=seu_client_secret_aqui
ALLOWED_PATHS=/home/usuario/Documentos,/home/usuario/Desktop
3

Iniciar o Servidor

Execute o servidor MCP:

python main.py

O servidor iniciara na porta 3000 por padrao

4

Configurar Claude Desktop

Adicione a configuracao ao Claude Desktop:

{
  "mcpServers": {
    "personal-mcp-server": {
      "command": "python",
      "args": ["/caminho/para/seu/main.py"],
      "env": {
        "GOOGLE_CLIENT_ID": "seu_client_id",
        "GOOGLE_CLIENT_SECRET": "seu_client_secret",
        "ALLOWED_PATHS": "/home/usuario/Documentos,/home/usuario/Desktop"
      }
    }
  }
}

Sistema de Extensibilidade

Como criar e adicionar novos modulos ao servidor

Template Basico de Modulo

import asyncio
from mcp import mcp
from typing import Optional

@mcp.tool()
async def minha_ferramenta(
    parametro1: str,
    parametro2: Optional[str] = None
) -> str:
    """Descricao da sua ferramenta.
    
    Args:
        parametro1: Descricao do primeiro parametro
        parametro2: Descricao do segundo parametro (opcional)
    
    Returns:
        Resultado da operacao
    
    Raises:
        ValueError: Quando os parametros sao invalidos
    """
    try:
        # Sua logica aqui
        resultado = f"Processando {parametro1}"
        if parametro2:
            resultado += f" com {parametro2}"
        
        return resultado
    except Exception as e:
        raise ValueError(f"Erro ao processar: {str(e)}")

Guia de Estilo de Codigo

Convencoes de Nomenclatura

  • Use snake_case para funcoes e variaveis
  • Use PascalCase para classes
  • Prefixe metodos privados com underscore
  • Use nomes descritivos e claros

Documentacao

  • Todas as funcoes devem ter docstrings
  • Use type hints em todos os parametros
  • Documente excecoes que podem ser levantadas
  • Inclua exemplos de uso quando necessario

Como Adicionar um Novo Modulo

  1. Crie um novo arquivo Python na pasta modules/
  2. Implemente suas ferramentas usando o decorador @mcp.tool()
  3. Adicione tratamento de erros apropriado
  4. Documente todas as funcoes com docstrings
  5. Importe o modulo no arquivo main.py
  6. Teste o modulo antes de usar em producao
